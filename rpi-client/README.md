# Raspberry Pi Client

This directory contains scripts and configuration for running the Infoskjermen display client on a Raspberry Pi.

## Modes
- **Color Mode**: Uses a standard browser in Kiosk mode.
- **Grayscale Mode**: Uses Puppeteer to render a high-contrast version of the frontend, screenshots it, and pushes it to the E-ink display.
